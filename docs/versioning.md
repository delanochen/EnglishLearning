# HomeLingua 版本策略

网页右下角始终显示当前应用版本，版本号以 `package.json` 为唯一来源。Docker 备份清单中的 `APP_VERSION` 应与它保持一致。

版本格式为 `主版本.功能版本.修订版本`：

- 修订版本（例如 `0.1.0 → 0.1.1`）：缺陷修复、文案调整、局部界面和体验优化，不改变主要使用流程。
- 功能版本（例如 `0.1.1 → 0.2.0`）：新增完整学习功能、重要流程调整、较大的数据模型或部署能力变化。
- 主版本（例如 `1.8.0 → 2.0.0`）：稳定版之后出现不兼容的架构、数据或部署方式升级。

每次发布到 `main` 前都必须按变更中最高级别的项目提升一次版本号；同一个版本号不得对应两次不同发布。
